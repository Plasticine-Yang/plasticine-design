# `@plasticine-design/spinner`

## Usage

```shell
npm i @plasticine-design/spinner @plasticine-design/styles
```

In your tsx:

```tsx
import { Spinner } from '@plasticine-design/spinner'
import '@plasticine-design/styles/components/spinner/index.scss'

const App = () => {
  return <Spinner />
}
```
